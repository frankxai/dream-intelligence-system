# Dream Intelligence System

> A SIP-conformant substrate for dream journaling, analysis, and pattern recognition — part of the [FrankX Intelligence Systems](https://github.com/frankxai/frankxai/blob/main/ECOSYSTEM.md) family.

## What this is

Dreams are signal. This system captures them, structures them, and surfaces patterns over time — recurring symbols, emotional arcs, and correlations with waking life. It is a domain vertical built on the shared FrankX spine, not a standalone app.

## Where it sits in the stack

```
Dream IS  →  composes  →  SIS (memory) + ACOS (skills) + SIP (standard)
```

- **Memory** persists through the [Starlight Intelligence System](https://github.com/frankxai/Starlight-Intelligence-System).
- **Skills/agents** run on [Agentic Creator OS](https://github.com/frankxai/agentic-creator-os).
- **Conformance** declared against [agentic-operating-system-standard](https://github.com/frankxai/agentic-operating-system-standard) (SIP).

## Planned modules

- `capture/` — fast morning dream capture (voice + text), timestamped to the SIS ledger
- `analyze/` — symbol extraction, recurring-motif detection, emotional-tone tagging
- `patterns/` — longitudinal views: themes over weeks/months, waking-life correlations
- `agent-skills/` — installable skills (`dream-capture`, `dream-analyze`) for any runtime

## Conventions

This repo follows the family naming: `dream-intelligence-system` (substrate) · `awesome-mind-agent-skills` (related index). Privacy-first — personal dream data stays in a private vault; only frameworks are public.
